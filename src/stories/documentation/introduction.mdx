# Welcome to Rawsome Components

## Installation 

This library can be installed via the cli using `npm`

```tsx
npm install rawsome-components
```
### Requirements 

Typescript, Tailwind and React.
## What is Rawsome components

Rawsome components is a reusable component library for projects that want simple to use and easy to style components.
